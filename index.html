IVED: Desktop software

Desktop software for inventory control with a sales module.

I worked as a freelance developer

The software integrates an inventory and sales module. Inventory is handled by sales or the stock module in which units leaving or entering are indicated. The user can generate reports of the registered trademarks, the detail of an invoice, all the invoices, make backups, upload a backup of said information, modify the credentials. Each registered product is assigned a code, from which a barcode is generated that can be exported as an image.
